# 🚀 Task Manager SaaS

## 📌 Description
Application fullstack permettant de gérer des tâches avec authentification.

## ⚙️ Stack technique
- Next.js (TypeScript)
- Supabase (PostgreSQL + Auth)
- Tailwind CSS

## ✨ Fonctionnalités
- Authentification utilisateur
- CRUD tâches
- Dashboard
- Filtrage
